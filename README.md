# Student Course Hub Portal

## Project Description
Student Course Hub is a web-based portal where students can log in, register, and manage their course information. The system allows students to access course details, view their dashboard, and interact with the platform easily.

## Technologies Used
- HTML5
- CSS3
- PHP
- MySQL
- Visual Studio Code

## Features
- Student registration
- Student login
- Dashboard page
- Database connection with MySQL
- Organized file structure

## Installation / Setup
1. Install WAMP or XAMPP.
2. Copy the project folder into the `www` directory.
3. Start Apache and MySQL.
4. Import the database file `student_portal.sql` into phpMyAdmin.
5. Open the browser and go to:
   http://localhost/student-portal
